Panduan Memulai

Langkah-langkah Instalasi Lokal

  1. 1. Kloning Repositori

    git clone <URL_REPOSITORI_ANDA>
cd idn-makerspace

  2. 2. Instal Dependensi

    npm install

  3. 3. Jalankan Server Pengembangan

    npm run dev

Gunakan skrip berikut untuk mengelola proyek Anda:

  • npm run dev: Menjalankan server pengembangan.
  • npm run build: Membangun aplikasi untuk produksi.
  • npm run lint: Menjalankan linter untuk memeriksa kualitas kode.
  • npm run preview: Menjalankan hasil build produksi secara lokal.
